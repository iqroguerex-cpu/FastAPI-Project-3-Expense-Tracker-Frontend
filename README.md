A **modern analytics dashboard** built with **Streamlit** for tracking and visualizing personal expenses.

The dashboard connects to a **FastAPI backend hosted on Render** and provides powerful insights including **expense analytics, category breakdowns, and spending trends**.

# ✨ Features

### 💰 Expense Tracking

* Add expenses quickly
* Delete expenses instantly
* View structured expense data

### 📊 Analytics Dashboard

Interactive visualizations including:

* Category spending pie chart
* Monthly spending bar chart
* Spending metrics

### 📈 Insights

Real-time statistics:

* Total expenses
* Total spending
* Average expense value

### 🔍 Filtering

Quick filtering by category.

### 📤 Data Export

Export all expenses to **CSV** for further analysis.

# ⚙️ Local Setup

Clone the repository

```
git clone https://github.com/iqroguerex-cpu/fastapi-project-3-expense-tracker-frontend
```

Navigate into the project

```
cd fastapi-project-3-expense-tracker-frontend
```

Install dependencies

```
pip install -r requirements.txt
```

Run the Streamlit app

```
streamlit run app.py
```
